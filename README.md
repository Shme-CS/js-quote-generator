# Quote Generator

A simple and elegant quote generator web application.

## Project Structure

```
js-quote-generator/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── .gitignore
└── README.md
```

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Setup

1. Clone the repository
2. Open `index.html` in your browser
3. Start generating quotes!
